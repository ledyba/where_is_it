# Where Is It?

地名を隠したストリートビューを10秒間眺めて、場所を当てるゲームです。

スマホのジャイロ機能に対応していて、スマホをかざすとそれに合わせてストリートビューも動きます。

目隠しをされて突然未知の場所に置き去りにされた人の気分になってプレイしてみてください。

[こちらで遊べます。](http://ledyba.org/utl/WhereIsIt/)
